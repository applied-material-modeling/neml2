Recall that all NEML2 models can be written in the following form
\f[
  y = f(x; p, b).
\f]

Defining a custom model therefore involves
- declaring the input variables \f$x\f$,
- declaring the output variables \f$y\f$, and
- declaring the parameters (and buffers, if any).
