# Solid mechanics

The solid-mechanics module is NEML2's largest physics catalog — a
collection of small `Model` building blocks for composing material
models for solids. Each sub-page below covers one sub-domain of
the theory: the physical motivation, the governing equations, at
least one worked input-file composition, and pointers to the
syntax-catalog entries for the individual types.
