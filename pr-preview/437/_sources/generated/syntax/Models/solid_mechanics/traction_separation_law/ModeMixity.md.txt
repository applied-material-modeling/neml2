# ModeMixity

**Source:** `models/solid_mechanics/traction_separation_law/ModeMixity.py`

Mode-mixity ratio $\beta = \delta_s / \delta_n^+$ in the opening
branch; $\beta = 0$ in compression. Uses a safe-divisor
`where`-and-detach pattern so the masked-off compression branch does not
trip on a division by zero.

## Inputs

`normal_separation` &mdash; `input` &middot; `Scalar` &middot; **required**
: Normal separation (typically the Macaulay-positive part of the normal jump)

`tangential_separation` &mdash; `input` &middot; `Scalar` &middot; **required**
: Tangential separation magnitude


## Outputs

`mode_mixity` &mdash; `output` &middot; `Scalar` &middot; **required**
: Mode-mixity ratio
