Detailed Description

Definition of a nonlinear system of equations, r(u) = 0.

Instead of directly defining a nonlinear system, we define the linearized system via its residual and Jacobian, evaluated at a given state u, the linearized system is usually expressed as dr(u)/du * du = -r(u). Rewriting this in the more familiar form Ax = b, we have A := dr(u)/du and b := -r(u).
