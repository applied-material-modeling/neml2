# Normality

**Source:** `models/solid_mechanics/plasticity/Normality.py`

Store the first derivatives of a scalar-valued function in given variables,
i.e. $u_i = \dfrac{f(\boldsymbol{v})}{v_i}$.

## Other options

`model` &mdash; `str` &middot; **required**
: The model which evaluates the scalar-valued function

`function` &mdash; `str` &middot; **required**
: Function to take derivative

`from` &mdash; `list` &middot; **required**
: Function arguments to take derivatives with respect to

`to` &mdash; `list` &middot; **required**
: Variables to store the first derivatives
