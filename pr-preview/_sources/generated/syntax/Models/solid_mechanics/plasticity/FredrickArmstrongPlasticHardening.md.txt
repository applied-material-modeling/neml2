# FredrickArmstrongPlasticHardening

**Source:** `models/solid_mechanics/plasticity/FredrickArmstrongPlasticHardening.py`

Map the flow rate (i.e., the consistency parameter in the KKT conditions)
to the rate of internal variables. This object defines the non-associative
Fredrick-Armstrong kinematic hardening. In the model, back stress is
directly treated as an internal variable. Rate of back stress is given as
$\dot{\boldsymbol{X}} = \left( \frac{2}{3} C \frac{\partial f}{\partial \boldsymbol{M}} - g \boldsymbol{X} \right) \dot{\gamma}$.$\frac{\partial f}{\partial \boldsymbol{M}}$
is the flow direction, $\dot{\gamma}$ is the flow rate, and $C$ and $g$ are material
parameters.

## Inputs

`flow_rate` &mdash; `input` &middot; `Scalar` &middot; **required**
: Flow rate

`flow_direction` &mdash; `input` &middot; `SR2` &middot; **required**
: Flow direction

`back_stress` &mdash; `input` &middot; `SR2` &middot; **required**
: Back stress


## Parameters

`C` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Kinematic hardening coefficient

`g` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Dynamic recovery coefficient
