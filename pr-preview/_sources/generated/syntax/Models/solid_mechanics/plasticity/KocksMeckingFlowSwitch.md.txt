# KocksMeckingFlowSwitch

**Source:** `models/solid_mechanics/plasticity/KocksMeckingFlowSwitch.py`

Switches between rate independent and rate dependent flow rules based on the value of the
Kocks-Mecking normalized activation energy.  For activation energies less than the threshold
use the rate independent flow rule, for values greater than the threshold use the rate
dependent flow rule.  This version uses a soft switch between the models, based on a tanh
sigmoid function.

## Inputs

`activation_energy` &mdash; `input` &middot; `Scalar` &middot; **required**
: The input name of the activation energy

`rate_independent_flow_rate` &mdash; `input` &middot; `Scalar` &middot; **required**
: Input name of the rate independent flow rate

`rate_dependent_flow_rate` &mdash; `input` &middot; `Scalar` &middot; **required**
: Input name of the rate dependent flow rate


## Outputs

`flow_rate` &mdash; `output` &middot; `Scalar` &middot; **required**
: Output name for the mixed flow rate


## Parameters

`g0` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Critical value of activation energy


## Other options

`sharpness` &mdash; `float` &middot; default `1.0`
: A steepness parameter that controls the tanh mixing of the models.  Higher values gives a sharper transition.
