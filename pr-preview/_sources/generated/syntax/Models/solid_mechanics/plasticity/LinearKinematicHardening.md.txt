# LinearKinematicHardening

**Source:** `models/solid_mechanics/plasticity/LinearKinematicHardening.py`

Map kinematic plastic strain to back stress following a linear
relationship, i.e., $\boldsymbol{X} = H \boldsymbol{K}_p$ where
$H$ is the hardening modulus.

## Inputs

`kinematic_plastic_strain` &mdash; `input` &middot; `SR2` &middot; **required**
: Kinematic plastic strain


## Outputs

`back_stress` &mdash; `output` &middot; `SR2` &middot; **required**
: Back stress


## Parameters

`hardening_modulus` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Hardening modulus
