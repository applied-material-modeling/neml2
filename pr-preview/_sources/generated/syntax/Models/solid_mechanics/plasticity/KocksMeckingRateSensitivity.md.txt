# KocksMeckingRateSensitivity

**Source:** `models/solid_mechanics/plasticity/KocksMeckingRateSensitivity.py`

Calculates the temperature-dependent rate sensitivity for a Perzyna-type model using the
Kocks-Mecking model.  The value is $n = \frac{\mu b^3}{k T A}$ with $\mu$ the
shear modulus, $b$ the Burgers vector, $k$ the Boltzmann constant, $T$
absolute temperature, and $A$ the Kocks-Mecking slope parameter.

## Inputs

`temperature` &mdash; `input` &middot; `Scalar` &middot; **required**
: Absolute temperature


## Outputs

`rate_sensitivity` &mdash; `output` &middot; `Scalar` &middot; **required**
: Output name of the rate sensitivity


## Parameters

`A` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: The Kocks-Mecking slope parameter

`shear_modulus` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: The shear modulus


## Other options

`k` &mdash; `float` &middot; **required**
: Boltzmann constant

`b` &mdash; `float` &middot; **required**
: The Burgers vector
