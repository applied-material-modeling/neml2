# ElasticStrainRate

**Source:** `models/solid_mechanics/crystal_plasticity/ElasticStrainRate.py`

Calculates the elastic strain rate as
$\dot{\varepsilon} = d - d^p - \varepsilon w + w \varepsilon$ where
$d$ is the deformation rate, $d^p$ is the plastic deformation
rate, $w$ is the vorticity, and $\varepsilon$ is the elastic
strain.

## Inputs

`elastic_strain` &mdash; `input` &middot; `SR2` &middot; **required**
: Name of the elastic strain

`deformation_rate` &mdash; `input` &middot; `SR2` &middot; **required**
: Name of the deformation rate

`vorticity` &mdash; `input` &middot; `WR2` &middot; **required**
: Name of the vorticity

`plastic_deformation_rate` &mdash; `input` &middot; `SR2` &middot; **required**
: Name of the plastic deformation rate


## Outputs

`elastic_strain_rate` &mdash; `output` &middot; `SR2` &middot; **required**
: Name of the elastic strain rate
