# VoceSingleSlipHardeningRule

**Source:** `models/solid_mechanics/crystal_plasticity/VoceSingleSlipHardeningRule.py`

Voce hardening for a SingleSlipStrength type model defined by
$\dot{\tau} = \theta_0 \left( 1 - \frac{\tau}{\tau_f} \right) \sum_{i=1}^{n_{slip}} \left| \dot{\gamma}_i \right|$
where
$\theta_0$ is the initial rate of work hardening, $\tau_f$ is
the saturated, maximum value of the slip system strength, and
$\dot{\gamma}_i$ is the slip rate on each system.

## Inputs

`slip_hardening` &mdash; `input` &middot; `Scalar` &middot; **required**
: Name of current values of slip hardening

`sum_slip_rates` &mdash; `input` &middot; `Scalar` &middot; **required**
: Name of tensor containing the sum of the slip rates


## Outputs

`slip_hardening_rate` &mdash; `output` &middot; `Scalar` &middot; **required**
: Name of the slip hardening rate


## Parameters

`initial_slope` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: The initial rate of hardening

`saturated_hardening` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: The final, saturated value of the slip system strength
