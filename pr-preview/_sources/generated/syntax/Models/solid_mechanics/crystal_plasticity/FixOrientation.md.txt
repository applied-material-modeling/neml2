# FixOrientation

**Source:** `models/solid_mechanics/crystal_plasticity/FixOrientation.py`

Swap the modified-Rodrigues representation to the shadow parameter set
when $\left\lVert r \right\rVert^2 \geq t$ (default threshold
$t = 1$), avoiding the $2\pi$ singularity. The shadow map is
$r^* = -r / \left\lVert r \right\rVert^2$.

See Banks, Matthew Jarrett, "Switching Methods for Three-Dimensional
Rotational Dynamics Using Modified Rodrigues Parameters." (2023).

## Inputs

`input` &mdash; `input` &middot; `Rot` &middot; **required**
: Name of input tensor of orientations to operate on.


## Outputs

`output` &mdash; `output` &middot; `Rot` &middot; **required**
: Name of output tensor


## Other options

`threshold` &mdash; `float` &middot; default `1.0`
: Threshold value for translating to the shadow parameters
