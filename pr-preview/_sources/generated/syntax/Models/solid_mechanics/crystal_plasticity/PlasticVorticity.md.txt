# PlasticVorticity

**Source:** `models/solid_mechanics/crystal_plasticity/PlasticVorticity.py`

Caclulates the plastic vorcitity as
$w^p = \sum_{i=1}^{n_{slip}} \dot{\gamma}_i Q \operatorname{skew}{\left(d_i \otimes n_i \right)} Q^T$
with
$d^p$ the plastic deformation rate, $\dot{\gamma}_i$ the slip
rate on the ith slip system, $Q$ the orientation, $d_i$ the slip
system direction, and $n_i$ is the slip system normal.

## Inputs

`orientation_matrix` &mdash; `input` &middot; `R2` &middot; **required**
: The name of the orientation matrix

`slip_rates` &mdash; `input` &middot; `Scalar` &middot; **required**
: The name of the tensor containg the current slip rates


## Outputs

`plastic_vorticity` &mdash; `output` &middot; `WR2` &middot; **required**
: The name of the plastic vorticity tensor


## Other options

`crystal_geometry` &mdash; `str` &middot; default `crystal_geometry`
: The name of the Data object containing the crystallographic information for the material
