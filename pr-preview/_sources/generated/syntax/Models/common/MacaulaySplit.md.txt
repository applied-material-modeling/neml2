# MacaulaySplit

**Source:** `models/common/MacaulaySplit.py`

Split a Scalar into its Macaulay (positive) and negative parts:
$\langle x \rangle_+ = \max(x, 0)$ and
$\langle x \rangle_- = x - \langle x \rangle_+$.

## Inputs

`from` &mdash; `input` &middot; `Scalar` &middot; **required**
: The Scalar to split


## Outputs

`to_positive` &mdash; `output` &middot; `Scalar` &middot; **required**
: Name of the Macaulay (positive) part output

`to_negative` &mdash; `output` &middot; `Scalar` &middot; **required**
: Name of the negative-part output
