# ScalarLinearCombination

**Source:** `models/common/LinearCombination.py`

Calculate linear combination of multiple Scalar tensors as
$u = w_i v_i + b$ (Einstein summation assumed), where $w_i$ are
the weights, and $v_i$ are the variables to be summed. $b$ is a
constant offset.

## Inputs

`from` &mdash; `input` &middot; `Scalar` &middot; **required**
: Scalar tensors to be summed


## Outputs

`to` &mdash; `output` &middot; `Scalar` &middot; **required**
: The sum


## Parameters

`weights` &mdash; `parameter` &middot; `Scalar` &middot; default `1`
: Per-input weight parameters. When the list is length 1, the single weight is broadcast across every from-variable.

`offset` &mdash; `parameter` &middot; `Scalar` &middot; default `0`
: The constant coefficient added to the final summation
