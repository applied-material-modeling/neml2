# ScalarForwardEulerTimeIntegration

**Source:** `models/common/ForwardEulerTimeIntegration.py`

Perform forward Euler time integration defined as
$s = s_n + (t - t_n) \dot{s}$, where $s$ is the variable being
integrated, $\dot{s}$ is the variable rate, and $t$ is time.
Subscripts $n$ denote quantities from the previous time step.

## Inputs

`time` &mdash; `input` &middot; `Scalar` &middot; default `t`
: Time


## Outputs

`variable` &mdash; `output` &middot; `Scalar` &middot; **required**
: Integrated variable


## Other options

`rate` &mdash; `str`
: Override name for the variable rate.
