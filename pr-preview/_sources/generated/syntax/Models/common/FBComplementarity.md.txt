# FBComplementarity

**Source:** `models/common/FBComplementarity.py`

If $a \ge 0, b \ge 0, ab = 0$ then the Fischer Burmeister (FB)
complementarity condition is:$r = a+b-\sqrt(a^2+b^2) = 0$.

## Inputs

`a` &mdash; `input` &middot; `Scalar` &middot; **required**
: First condition

`b` &mdash; `input` &middot; `Scalar` &middot; **required**
: Second condition


## Outputs

`complementarity` &mdash; `output` &middot; `Scalar` &middot; **required**
: The Fischer Burmeister complementarity condition


## Other options

`a_inequality` &mdash; `str` &middot; default `LE`
: Type of the inequality for the first variable a.

`b_inequality` &mdash; `str` &middot; default `GE`
: Type of inequality for the second variable b.
