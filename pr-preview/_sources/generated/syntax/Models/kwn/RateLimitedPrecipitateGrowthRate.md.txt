# RateLimitedPrecipitateGrowthRate

**Source:** `models/kwn/RateLimitedPrecipitateGrowthRate.py`

Compute the rate-limited precipitate growth rate for a single species.

## Inputs

`current_concentration` &mdash; `input` &middot; `Scalar` &middot; **required**
: Current concentration in solution


## Outputs

`growth_rate` &mdash; `output` &middot; `Scalar` &middot; **required**
: Precipitate growth rate per size bin


## Parameters

`radius` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Precipitate radius per size bin

`equilibrium_concentration` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Equilibrium concentration in solution

`concentration_difference` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Concentration difference between precipitate and equilibrium

`diffusivity` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Species diffusivity in solution
