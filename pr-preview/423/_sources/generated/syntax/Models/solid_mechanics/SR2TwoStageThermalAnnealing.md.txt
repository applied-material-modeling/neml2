# SR2TwoStageThermalAnnealing

**Source:** `models/solid_mechanics/TwoStageThermalAnnealing.py`

SR2-valued two-stage thermal annealing. Mirrors ``TwoStageThermalAnnealing<SR2>``.

## Inputs

`base_rate` &mdash; `input` &middot; `SR2` &middot; **required**
: Base hardening rate

`base` &mdash; `input` &middot; `SR2` &middot; **required**
: Underlying base hardening variable

`temperature` &mdash; `input` &middot; `Scalar` &middot; **required**
: Temperature


## Outputs

`modified_rate` &mdash; `output` &middot; `SR2` &middot; **required**
: Output for the modified hardening rate.


## Parameters

`T1` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: First stage annealing temperature

`T2` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Second stage annealing temperature

`tau` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Recovery rate for second stage annealing.
