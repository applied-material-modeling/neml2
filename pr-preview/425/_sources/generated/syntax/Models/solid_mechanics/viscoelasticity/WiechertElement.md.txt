# WiechertElement

**Source:** `models/solid_mechanics/viscoelasticity/WiechertElement.py`

Wiechert (generalized Maxwell) viscoelastic model with two Maxwell branches in
parallel with an equilibrium spring. Total stress is
$\boldsymbol{\sigma} = (E_\infty + E_1 + E_2) \boldsymbol{\varepsilon} - E_1 \boldsymbol{\varepsilon}_{v,1} - E_2 \boldsymbol{\varepsilon}_{v,2}$,
with
each viscous strain evolving according to
$\dot{\boldsymbol{\varepsilon}}_{v,i} = E_i (\boldsymbol{\varepsilon} - \boldsymbol{\varepsilon}_{v,i}) / \eta_i$.

## Inputs

`strain` &mdash; `input` &middot; `SR2` &middot; **required**
: Total strain

`viscous_strain_1` &mdash; `input` &middot; `SR2` &middot; **required**
: Viscous strain in the first Maxwell branch

`viscous_strain_2` &mdash; `input` &middot; `SR2` &middot; **required**
: Viscous strain in the second Maxwell branch


## Outputs

`stress` &mdash; `output` &middot; `SR2` &middot; **required**
: Total stress


## Parameters

`equilibrium_modulus` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Equilibrium spring modulus

`modulus_1` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Spring modulus of the first Maxwell branch

`viscosity_1` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Dashpot viscosity of the first Maxwell branch

`modulus_2` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Spring modulus of the second Maxwell branch

`viscosity_2` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Dashpot viscosity of the second Maxwell branch
