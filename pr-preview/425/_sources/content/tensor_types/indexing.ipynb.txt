    "(tensor-types-indexing)=\n",
    "# Indexing\n",
    "\n",
    "Indexing a wrapper follows the same principle as\n",
    "[](tensor-types-region-views): you index *through a region view*, so a\n",
    "slice or integer index addresses one region's axes and the other\n",
    "regions pass through untouched. The index syntax itself is ordinary\n",
    "NumPy/torch indexing — slices, integers, `None`, `Ellipsis`; the\n",
    "authoritative reference for that syntax is the\n",
    "[NumPy indexing guide](https://numpy.org/doc/stable/user/basics.indexing.html).\n",
    "What NEML2 adds is the region scoping and the metadata bookkeeping that\n",
    "comes with it."

    "## Region-scoped indexing\n",
    "\n",
    "A key on `t.dynamic_batch` or `t.sub_batch` addresses only that\n",
    "region's axes; everything before and after passes through as `:`. The\n",
    "wrapper's `sub_batch_ndim` is adjusted automatically — an integer index\n",
    "drops an axis, `None` inserts one:"

    "from neml2.types import Scalar\n",
    "\n",
    "V = Scalar(torch.arange(5.0)).sub_batch.retag(1)  # length-5 sub-batch axis\n",
    "print(\n",
    "    \"V.sub_batch[0]   ->\",\n",
    "    tuple(V.sub_batch[0].data.shape),\n",
    "    \"sub_ndim\",\n",

    "## Base components\n",
    "\n",
    "The base region of a fixed-base primitive is *not* indexable through a\n",
    "view — its shape and packing are part of the type, so reaching into it\n",
    "by raw position would bypass the convention (the Mandel √2 on an `SR2`,\n",
    "for instance). To pull a component out as a `Scalar`, use the\n",
    "`vec_component` free function:"

    "## How this differs from plain torch indexing\n",
    "\n",
    "On a raw `torch.Tensor` you index the *flat* shape and carry the\n",
    "responsibility of knowing which axis is which. Through a region view you\n",
    "index *one region*, the rest pass through automatically, and the\n",
    "returned wrapper's region metadata is updated for you — so a sub-batch\n",
    "slice stays a valid sub-batch, and a dropped axis decrements\n",
    "`sub_batch_ndim` instead of silently misaligning later operations. For\n",
    "the index *syntax* (basic slicing, integer arrays, `None`, `Ellipsis`),\n",
    "the [NumPy indexing guide](https://numpy.org/doc/stable/user/basics.indexing.html)\n",
    "is the reference."
