# LinearIsotropicStrainEnergyDensity

**Source:** `models/phase_field_fracture/LinearIsotropicStrainEnergyDensity.py`

Calculates elastic strain energy density based on linear elastic isotropic response

## Inputs

`strain` &mdash; `input` &middot; `SR2` &middot; **required**
: Elastic strain


## Outputs

`active_strain_energy_density` &mdash; `output` &middot; `Scalar` &middot; **required**
: Active part of the strain energy density

`inactive_strain_energy_density` &mdash; `output` &middot; `Scalar` &middot; **required**
: Inactive part of the strain energy density


## Other options

`coefficients` &mdash; `list` &middot; **required**
: Coefficients used to define the elasticity tensor

`coefficient_types` &mdash; `list` &middot; **required**
: Types for each parameter, options are: INVALID, P_WAVE_MODULUS, POISSONS_RATIO, YOUNGS_MODULUS, SHEAR_MODULUS, BULK_MODULUS, LAME_LAMBDA

`decomposition` &mdash; `str` &middot; default `NONE`
: Strain energy density decomposition types, options are: VOLDEV, SPECTRAL, NONE
