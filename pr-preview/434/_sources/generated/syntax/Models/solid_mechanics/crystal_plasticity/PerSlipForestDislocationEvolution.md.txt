# PerSlipForestDislocationEvolution

**Source:** `models/solid_mechanics/crystal_plasticity/PerSlipForestDislocationEvolution.py`

Standard forest hardening model per slip system defined by
$\dot{\rho}_i = (k_1 \sqrt{\rho_i} - k_2 \rho_i) \left| \dot{\gamma}_i \right|$.

## Inputs

`dislocation_density` &mdash; `input` &middot; `Scalar` &middot; **required**
: Per-slip dislocation density

`slip_rates` &mdash; `input` &middot; `Scalar` &middot; **required**
: Per-slip system slip rates


## Outputs

`dislocation_density_rate` &mdash; `output` &middot; `Scalar` &middot; default `dislocation_density_rate`
: Optional name for the derived output (defaults to 'dislocation_density_rate', i.e. the 'dislocation_density' variable name with the '_rate' suffix).


## Parameters

`k1` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Hardening coefficient

`k2` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Recovery coefficient
