(solvers-Newton)=
# Newton

**Source:** `solvers/newton.py`

The standard Newton-Raphson solver which always takes the 'full' Newton step.

## Other options

`linear_solver` &mdash; `str`
: The linear solver to use within the nonlinear solver

`abs_tol` &mdash; `float` &middot; default `1e-10`
: Absolute tolerance in the convergence criteria

`rel_tol` &mdash; `float` &middot; default `1e-08`
: Relative tolerance in the convergence criteria

`max_its` &mdash; `int` &middot; default `25`
: Maximum number of iterations allowed before issuing an error/exception

`verbose` &mdash; `bool` &middot; default `False`
: Print the Newton convergence history (per-iteration residual norms, including any line-search sub-iterations) after each solve
