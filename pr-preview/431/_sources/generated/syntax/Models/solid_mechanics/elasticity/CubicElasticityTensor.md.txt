# CubicElasticityTensor

**Source:** `models/solid_mechanics/elasticity/CubicElasticityTensor.py`

This class defines a cubic anisotropic elasticity tensor using three
parameters.  Various options are available for which three parameters to
provide.

## Outputs

`C` &mdash; `output` &middot; `SSR4`
: Cubic elasticity tensor


## Other options

`coefficients` &mdash; `list` &middot; **required**
: Coefficients used to define the elasticity tensor

`coefficient_types` &mdash; `list` &middot; **required**
: Types for each parameter, options are: INVALID, P_WAVE_MODULUS, POISSONS_RATIO, YOUNGS_MODULUS, SHEAR_MODULUS, BULK_MODULUS, LAME_LAMBDA
