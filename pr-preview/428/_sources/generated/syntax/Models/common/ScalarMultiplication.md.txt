# ScalarMultiplication

**Source:** `models/common/ScalarMultiplication.py`

Calculate the product of multiple Scalar variables with a constant scaling
coefficient. Using reciprocal, one can have the reciprocity of each variable

## Inputs

`from` &mdash; `input` &middot; `Scalar` &middot; **required**
: Scalar variables to be multiplied


## Outputs

`to` &mdash; `output` &middot; `Scalar` &middot; **required**
: The multiplicative product


## Parameters

`scaling` &mdash; `parameter` &middot; `Scalar` &middot; default `1`
: The scaling coefficient to multiply to the final product


## Other options

`reciprocal` &mdash; `list` &middot; default `False`
: List of boolens, one for each variable, in which the reciprocity of the corresponding variable is taken. When the length of this list is 1, the same reciprocal condition applies to all variables.
