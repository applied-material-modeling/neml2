# LinearIsotropicElasticJ2TrialStressUpdate

**Source:** `models/solid_mechanics/plasticity/LinearIsotropicElasticJ2TrialStressUpdate.py`

Update the trial stress under the assumptions of J2 plasticity and
isotropic linear elasticity.

This allows the construction of fully scalar return mapping models for
isotropic materials.

## Inputs

`elastic_trial_stress` &mdash; `input` &middot; `Scalar` &middot; **required**
: Initial trial stress assuming a purely elastic step

`equivalent_plastic_strain` &mdash; `input` &middot; `Scalar` &middot; **required**
: Current guess for the equivalent plastic strain


## Outputs

`updated_trial_stress` &mdash; `output` &middot; `Scalar` &middot; **required**
: Trial stress corrected for the current increment of plastic deformation


## Other options

`coefficients` &mdash; `list` &middot; **required**
: Coefficients used to define the elasticity tensor

`coefficient_types` &mdash; `list` &middot; **required**
: Types for each parameter, options are: INVALID, P_WAVE_MODULUS, POISSONS_RATIO, YOUNGS_MODULUS, SHEAR_MODULUS, BULK_MODULUS, LAME_LAMBDA
