# OrthotropicLinearTraction

**Source:** `models/solid_mechanics/traction_separation_law/OrthotropicLinearTraction.py`

Orthotropic linear-elastic interface traction:
$T_n = K_n \delta_n^\text{sep}$, $T_{si} = K_t \delta_{si}$.
If `normal_penetration` is supplied,
$K_\text{pen} \delta_n^\text{pen}$ is added to $T_n$ as a
penalty term resisting interpenetration (`penalty_stiffness` becomes
required); otherwise interpenetration produces zero normal traction.

## Inputs

`normal_separation` &mdash; `input` &middot; `Scalar` &middot; **required**
: Normal separation (typically the Macaulay-positive part of the normal jump)

`normal_penetration` &mdash; `input` &middot; `Scalar`
: Optional normal penetration (typically the Macaulay-negative part of the normal jump). When supplied, the penalty stiffness times this is added to the normal traction as a penalty term resisting interpenetration. Requires `penalty_stiffness` to be supplied as well.

`tangential_separation_1` &mdash; `input` &middot; `Scalar` &middot; **required**
: First tangential separation

`tangential_separation_2` &mdash; `input` &middot; `Scalar` &middot; **required**
: Second tangential separation


## Outputs

`traction` &mdash; `output` &middot; `Vec` &middot; **required**
: Traction Vec


## Parameters

`normal_stiffness` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Normal stiffness

`tangential_stiffness` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Tangential stiffness (isotropic)

`penalty_stiffness` &mdash; `parameter` &middot; `Scalar` &middot; default `0.0`
: Penalty stiffness used to resist interpenetration. Required when `normal_penetration` is supplied; ignored otherwise.
