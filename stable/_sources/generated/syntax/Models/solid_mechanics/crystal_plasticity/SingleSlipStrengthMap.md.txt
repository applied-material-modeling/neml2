# SingleSlipStrengthMap

**Source:** `models/solid_mechanics/crystal_plasticity/SingleSlipStrengthMap.py`

Calculates the slip system strength for all slip systems as
$\hat{\tau}_i = \bar{\tau} + \tau_0$ where $\hat{\tau}_i$ is
the strength for slip system i, $\bar{\tau}$ is an evolving slip
system strength (one value of all systems), defined by another object, and
$\tau_0$ is a constant strength.

## Inputs

`slip_hardening` &mdash; `input` &middot; `Scalar` &middot; **required**
: The name of the evolving, scalar strength


## Outputs

`slip_strengths` &mdash; `output` &middot; `Scalar` &middot; **required**
: Name of the slip system strengths


## Parameters

`constant_strength` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: The constant slip system strength


## Other options

`crystal_geometry` &mdash; `str` &middot; default `crystal_geometry`
: The name of the Data object containing the crystallographic information for the material
