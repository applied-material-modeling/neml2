(solvers-NewtonWithLineSearch)=
# NewtonWithLineSearch

**Source:** `solvers/newton_linesearch.py`

The Newton-Raphson solver with line search.

## Other options

`linear_solver` &mdash; `str`
: The linear solver to use within the nonlinear solver

`abs_tol` &mdash; `float` &middot; default `1e-10`
: Absolute tolerance in the convergence criteria

`rel_tol` &mdash; `float` &middot; default `1e-08`
: Relative tolerance in the convergence criteria

`max_its` &mdash; `int` &middot; default `25`
: Maximum number of iterations allowed before issuing an error/exception

`substep_del_tol` &mdash; `float` &middot; default `1e-06`
: Relative Newton-step tolerance gating the relative convergence branch of the masked (substepping) solve: a sub-step is accepted via rel_tol only if its step also satisfies ||du|| < substep_del_tol*(||u||+abs_tol). This bisects instead of freezing a still-moving, non-physical iterate that merely dips under a loose rel_tol threshold when the predictor residual is inflated. Set large to recover the pure relative test. Ignored by the single-shot (non-substepping) solve.

`linesearch_type` &mdash; `str` &middot; default `BACKTRACKING`
: The type of linesearch used. Options are BACKTRACKING and STRONG_WOLFE.

`max_linesearch_iterations` &mdash; `int` &middot; default `10`
: Maximum allowable linesearch iterations. No error is produced upon reaching the maximum number of iterations, and the scale factor in the last iteration is used to scale the step.

`linesearch_cutback` &mdash; `float` &middot; default `2.0`
: Linesearch cut-back factor when the current scale factor cannot sufficiently reduce the residual.

`linesearch_stopping_criteria` &mdash; `float` &middot; default `0.001`
: The linesearch tolerance slightly relaxing the definition of residual decrease

`check_negative_criterion` &mdash; `bool` &middot; default `False`
: Whether to check if the threshold used in the convergence criterion for line search becomes negative. If true, and a negative value is detected, a warning message is printed.
