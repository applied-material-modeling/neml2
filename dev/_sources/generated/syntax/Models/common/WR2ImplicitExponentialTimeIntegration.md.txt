# WR2ImplicitExponentialTimeIntegration

**Source:** `models/common/WR2ImplicitExponentialTimeIntegration.py`

Define the implicit discrete exponential time integration residual of a
rotation variable. The residual can be written as
$r = s - \exp\left[ (t-t_n)\dot{s}\right] \circ s_n$, where
$\circ$ denotes the rotation operator.

## Inputs

`variable` &mdash; `input` &middot; `MRP` &middot; **required**
: Integrated rotation

`time` &mdash; `input` &middot; `Scalar` &middot; default `t`
: Time


## Outputs

`variable_residual` &mdash; `output` &middot; `MRP` &middot; default `variable_residual`
: Optional name for the derived output (defaults to 'variable_residual', i.e. the 'variable' variable name with the '_residual' suffix).


## Other options

`rate` &mdash; `str`
: Override name for the variable rate.
