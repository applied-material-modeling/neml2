# PowerLawIsotropicHardeningStaticRecovery

**Source:** `models/solid_mechanics/plasticity/PowerLawIsotropicHardeningStaticRecovery.py`

This particular model implements a power law recovery of the type
$\dot{k} = -\left(\frac{\lVert k \rVert}{\tau}\right)^{n-1} \frac{k}{\tau}$

## Inputs

`isotropic_hardening` &mdash; `input` &middot; `Scalar` &middot; **required**
: Isotropic hardening variable


## Outputs

`isotropic_hardening_rate` &mdash; `output` &middot; `Scalar` &middot; default `isotropic_hardening_rate`
: Optional name for the derived output (defaults to 'isotropic_hardening_rate', i.e. the 'isotropic_hardening' variable name with the '_rate' suffix).


## Parameters

`tau` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Recovery rate

`n` &mdash; `parameter` &middot; `Scalar` &middot; **required**
: Recovery exponent
