CurrentConcentration¶

Source: models/kwn/CurrentConcentration.py

Compute the current matrix concentration from precipitate volume fractions.

For \(N\) precipitate species with volume fractions \(f_i\) and (constant) precipitate concentrations \(x_i^p\), the matrix concentration is

\[ x = \frac{x_0 - \sum_i f_i x_i^p}{1 - \sum_i f_i} \]

where \(x_0\) is the initial concentration in solution.

precipitate_volume_fractions — input · Scalar · required

Precipitate volume fraction variables

Outputs¶

current_concentration — output · Scalar · required

Current concentration in solution

Parameters¶

Initial concentration in solution

Precipitate concentrations
